# fastapi-media-stuff

A repo contains fastapi media convertion and stuff yeah

## What it should have

1. image resizer
2. file converter
3. audio slicer
4. audio merger  
5. level image renderer
that's about it

## setup

1. `poetry install`
2. `poetry shell`
3. configure your .env

```bash
REDIS_URI=your redis uri
```

4. `uvicorn app:app {any config as you desire}`
